Background con photoshop

Fondo de Pantalla Espacial

Este proyecto consiste en la creación de un fondo de pantalla utilizando técnicas avanzadas de edición en Photoshop. El diseño presenta una escena interestelar con colores vibrantes y elementos inspirados en el espacio.

Vectorización de personaje

Vectorización Artística

En este proyecto se realiza la vectorización detallada de un personaje, transformando una imagen rasterizada en un arte vectorial de alta calidad. Se emplean técnicas de ilustración digital para resaltar los detalles y lograr un resultado visual impactante.

Teoría del color

Este proyecto explora la aplicación de la teoría del color en el diseño gráfico. Se toma una misma imagen vectorizada y se presentan seis variaciones, cada una utilizando una paleta de colores diferente. Las paletas incluyen monocromática, triada, complementaria, cuadrada y análoga, demostrando cómo la selección adecuada de colores puede influir en la percepción visual de una composición.
